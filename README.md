# Javascript V8 library builds

Build V8 v9.4 static and shared libraries.
This repo contains release and debug builds of the javascript v8 libraries for the following platforms.

- Linux x64 built on Debian 10 (Buster)
- Linux arm64 built on Debian 10 (Buster)
- MacOS x64(Intel) built on MacOS 13 (Ventura)
- MacOS arm64(Apple Silicon) built on MacOS 14 (Sonoma)
